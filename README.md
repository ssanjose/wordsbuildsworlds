Words Builds Worlds(WBW - webew) is an experimental method to revise learning by helping a person build a new perception of reality.

> [!CAUTION]
> WBW is dependent on the author's biased view of the world and how learning should be approached. Please use with caution. Mental models can be dangerous pits if not tread on carefully.

## Foundations
WBW relies on two foundations:
- <strong>Words create worlds</strong>: Language(signs, symbols, letters) influences thought and helps speakers perceive and categorize reality.
- <strong>Insights are everywhere</strong>: Learning is derivation of connections between concepts/words. That we can understand a concept through a connection to another.

This allows one to do top-down learning by using a selected word to filter reality.

## Goals
WBW is built to:
- Help create a structured way of passing/learning wisdom.
- Help create new perceptions of reality and enjoyment.
- Facilitate critical thinking for readers

## Caveats
- WBW is an experimental method to create mental models. Please use with caution.
- Academic learning and apprenticeship are great ways to learn concrete and actionable knowledge. WBW does not replace these kind of method, but plan to enhance it.

## Contributing
First, you need to have a github account to clone or fork this repository. To run this application, you need npm and node installed.

Then, running locally
```cmd
cd web
npm run dev
```

When you want to make a change in the master repository(which the deployments are based on), you need to fork it, submit the change via a pull request(PR). I'll review it then merge and deploy to the main site.
