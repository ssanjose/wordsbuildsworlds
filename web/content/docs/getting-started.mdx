---
title: Getting Started
description: We get started by following the for steps.
---

1. <strong>Pick an existing problem; don't create one.</strong>: Write down the situation you are trying to understand.
2. <strong>Select an Anchor Word</strong>: Browse the WBW directory for sample words and pick one.
3. <strong>Apply the Prism</strong>: Ask. Apply the problem and filter it through the anchor word.
4. <strong>Learn the Lesson</strong>: Find conclusions. It doesn't neccessarily need to fill the whole story, just provide insights to learn bit by bit, step by step.
5. <strong>Pick Another Anchor Word</strong>: If one isn't enough, pick another anchor word to find a new angle to the problem. Why is that? How did that happen? 

Over time, the problem will become a complete puzzle with each anchor word giving a piece.
